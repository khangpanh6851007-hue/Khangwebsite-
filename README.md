# TÙNG QUAY HEO - v17.0 Pro

Hệ thống website thương mại điện tử và cẩm nang ẩm thực chuyên nghiệp, hỗ trợ đồng bộ dữ liệu thời gian thực qua Firebase Realtime Database.

## Tính năng nổi bật
* **Quản lý sản phẩm & Bảng giá:** Tự động đồng bộ các mặt hàng heo quay, heo sữa, heo nguyên con với giá tiền chuẩn xác.
* **Đăng bài viết mới (Blog):** Cho phép người dùng dễ dàng chia sẻ bí quyết nấu nướng, cẩm nang ẩm thực trực tiếp lên giao diện website.
* **Cập nhật thông tin nhanh chóng:** Tích hợp gọi điện nhanh, đặt lịch cúng/lễ và tính năng săn voucher hấp dẫn.
* **Cơ sở dữ liệu đám mây:** Sử dụng Firebase Realtime Database v8.10.1 giúp lưu trữ dữ liệu thông suốt và liên tục.

## Hướng dẫn cài đặt & Chạy ứng dụng
1. Tải hoặc sao chép mã nguồn vào môi trường chạy cục bộ (như Pydroid 3 trên Android hoặc máy chủ Local Server).
2. Mở file HTML chính thông qua trình duyệt web.
3. Đảm bảo thiết bị có kết nối Internet để nạp các thư viện giao diện Tailwind CSS, FontAwesome và SDK Firebase.
